Syncany Microsoft OneDrive Plugin
----------------------
This plugin uses a **Microsoft OneDrive folder** as a storage backend for [Syncany](http://www.syncany.org). The plugin can be installed in Syncany using the `sy plugin install` command. For further information about the usage, please refer to the central **[wiki page](https://github.com/binwiederhier/syncany/wiki)**.

For plugin development, please refer to the [plugin development wiki page](https://github.com/binwiederhier/syncany/wiki/Plugin-development).

### Please Note: The pungin is in development and not ready for any use, yet
